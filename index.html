This is an early-stage framework for implementing Azure Functions in Rust

Azure Functions for Rust is a project that enables authoring functions in Rust that can be easily deployed to Azure Functions.

The project is currently under heavy development as it marches towards a 1.0.0 stable release.

Check out the GitHub project page for more information.
